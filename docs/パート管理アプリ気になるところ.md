# パート管理アプリ気になるところ

## 新規社員登録

- 雇用開始日と雇用終了日は近い場所がいい
- 役職、雇用形態は不要
- 勤務時間（時間）を時間帯で入力したい（数字しか入力できない）
- 勤務時間（時間）は複数ある人もいる
- 休憩時間は時間数ではなく時間帯入力したい
- 勤務日数（週）週○日とは限らない、月○日とかシフトとかある
- 勤務場所も複数ある人いる
- 残業時給は不要
- 有休休暇付与の基準日を入れてほしい
- 交通費情報には往復金額も入れたい
- 月額交通費は定期
- 上限金額も入れたい
- ルートごとに入力できないか（電車、バス分けて）
- 連絡先情報は統括としては不要だけど、現場ではあった方がいい？
  - →結構な個人情報になる
- 銀行口座情報は不要かも？
- エンター押すと保存されてしまう
  - →次の項目に行くとか
- 勤務条件、保険・福利厚生が保存できない（入力しても、保存ボタン押すと消える）
- 書類管理はまだ入力できない？
- 雇用契約書を作成する想定なら、足りないところがある
  - →契約更新の有無、有期契約の基準日、業務の内容など
- 情報を蓄積したい
- 部署を部門コードごとにした方がいいかも？
- 備考欄がほしい

## 社員一覧表示

- 一覧で出力したい
- 勤務場所で検索したい（雇用形態は不要）
- ステータスが分かるといい（現場作成中、事業部長押印待ち、総務提出済、本人返却済とかいろいろ）
  - →一覧でチェックした人のステータスが変わるといい

- 雇用期間終了日が迫ってるアラート（雇用契約書出力したらアラート解除）
- 社員番号-枝番を雇用契約書に載せられないか（出力のたびに枝番増える）
