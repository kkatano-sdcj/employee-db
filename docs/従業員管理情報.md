# `雇用情報`ページに表示するデータ項目

- 社員番号
- 氏名
- 部門
- 契約番号
- 入社日
- 雇用期間
- 退社日

# `勤務情報`ページに表示するデータ項目

- 契約書有給
- 勤務時間
- 休憩時間
- 勤務日数/週
- 勤務場所
- 業務内容

# `給与・手当`に表示する項目
- 時給
- 残業時給
- 最寄り駅
- 交通費（片道/往復）
- 控除申告書（甲乙）
- 雇用保険（加入/未加入）
- 雇用保険書提出
- 社会保険（加入/未加入）

# `書類`に表示する項目
- 保険証授
- 雇用契約書他管理へ提出(日付)
- 本人へ返却
- 満了通知書発効
- 退職届提出
- 返却
 - 保険証
 - セキュリティカード
